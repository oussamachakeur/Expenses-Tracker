## How to Use

1. **Add an Expense**: Fill in the form with expense details and click "Add Expense"
2. **Filter Expenses**: Use the dropdown filters to view specific periods
3. **Sort Expenses**: Choose how to sort your expense list
4. **Delete Expense**: Click the trash icon to remove an expense
5. **View Chart**: Check the monthly breakdown chart for visual insights
